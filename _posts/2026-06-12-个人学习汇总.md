## Hey

1.nginx里面配置127.0.0.1和配置本机公网ip的效果是不一样的,配置公网ip的端口你必须要打开防火墙

2.单机的session用cookie来实现,集群的session用redis来实现

3.nginx的配置location /search/ {
    root html;
    proxy_pass http://Apr1stserveres;
    index index.html index.htm;
} 和 
location / {
    root html;
    proxy_pass http://Apr1stserveres;
    index index.html index.htm;
} 和
location /search/ {
    root html;
    proxy_pass http://Apr1stserveres/;
    index index.html index.htm;
} 的区别在于什么呢? 其一,1和2匹配模式不同,同样命中时1长所以转发到1; 其二,1和3匹配模式也不同,/search无法被1匹配, http://Apr1stserveres/会把/search去掉,但是http://Apr1stserveres会保留/search/
