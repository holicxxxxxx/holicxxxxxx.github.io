## Hey
CORS 就是跨域资源共享。
跨域问题，本质是：

浏览器不允许一个网页随便请求另一个“不同源”的接口。

何为同源:
协议 + 域名/IP + 端口
当以上三个任意一个不一致的情况下,视为有跨域问题出现
诸如一下问题:

| 前端页面地址                  | 请求接口地址                 | 是否跨域 | 原因       |
| ----------------------- | -------------------------------- | ---- | -------- |
| `http://localhost:8080` | `http://localhost:8080/books/1`  | 否    | 完全一样     |
| `http://localhost:8080` | `http://localhost:8081/books/1`  | 是    | 端口不同     |
| `http://localhost:8080` | `http://127.0.0.1:8080/books/1`  | 是    | 域名/IP 不同 |
| `http://localhost:8080` | `https://localhost:8080/books/1` | 是    | 协议不同     |
| `http://aaa.com`        | `http://bbb.com/api`             | 是    | 域名不同     |
